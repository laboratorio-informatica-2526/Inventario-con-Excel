# Inventario-con-Excel
Modulo pratico di Excel dedicato alla creazione e gestione di un inventario digitale. Il percorso accompagna passo dopo passo dalla progettazione del file all’inserimento corretto dei dati, fino al controllo di scadenze e all’analisi tramite filtri, formule e formattazione condizionale. Obiettivo: autonomia e consapevolezza nell’uso dei dati.
